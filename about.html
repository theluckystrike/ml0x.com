About ml0x

ml0x.com is a free, open-source machine learning reference tool. Select any core ML algorithm and instantly generate a formatted cheat sheet with the key formulas, hyperparameter guides, pros and cons, and working Python code snippets.

Every cheat sheet is designed to be printable (use the Print button for a clean layout) or copyable as Markdown for your notes, documentation, or team wiki.

What's Covered

  • Convolutional Neural Networks (CNN)
  • Recurrent Networks (RNN, LSTM, GRU)
  • Transformers and Self-Attention
  • Decision Trees and Ensemble Methods
  • Support Vector Machines
  • K-Nearest Neighbors
  • Linear and Logistic Regression
  • Gradient Descent Variants (SGD, Adam, AdamW)
  • Regularization (L1, L2, Dropout, BatchNorm)

Why We Built This

Most ML references are either too academic (30-page papers) or too shallow (one-paragraph summaries). ml0x sits in the middle: enough depth to be useful during implementation, concise enough to fit on a single screen.
