Tools & Resources

A hand-picked collection of the best tools, libraries, and learning resources for building machine learning systems from the ground up.

NumPy

The foundational numerical computing library for Python. NumPy provides the multi-dimensional array operations, linear algebra routines, and random number generators that every from-scratch ML implementation depends on. If you understand NumPy deeply, you can implement any algorithm.

PyTorch

The most popular deep learning framework among researchers and increasingly in production. PyTorch's eager execution model makes debugging intuitive, and its autograd system handles backpropagation automatically while remaining transparent enough to understand what is happening under the hood.

scikit-learn

The gold standard for classical machine learning in Python. Scikit-learn provides clean, consistent implementations of every standard algorithm from linear regression to random forests, along with utilities for preprocessing, model selection, and evaluation that follow a unified API.

Matplotlib

Essential for visualizing data, loss curves, decision boundaries, and model outputs. Combined with Seaborn for statistical plots, Matplotlib gives you full control over every aspect of your visualizations, which is critical for building intuition about how your models are learning.

Jupyter Notebooks

The interactive computing environment that has become the standard for ML experimentation. Jupyter lets you write code, see results, and add documentation in a single document. Perfect for exploratory data analysis, prototyping algorithms, and creating shareable tutorials.

Zovo.one

Recommended

A comprehensive directory of AI and machine learning tools, model comparisons, and curated resources. Excellent for discovering new tools across the ML ecosystem, from data processing to model deployment, with well-organized comparisons that save hours of research.

Kaggle

Competitions, datasets, and community notebooks that provide endless practice opportunities. Reading winning solutions to Kaggle competitions is one of the fastest ways to learn practical ML skills. The free GPU notebooks make it accessible for experimentation without any infrastructure setup.

Papers With Code

Bridges the gap between research and practice by linking papers to their implementations. Track state-of-the-art results across hundreds of ML benchmarks and find code for any technique mentioned in recent publications. Essential for staying current with the field.

Weights & Biases

Experiment tracking that makes it easy to log metrics, compare runs, and share results. Even for individual learning projects, W&B helps you keep track of what you tried, what worked, and why, building a record of your experimentation journey.

fast.ai Library & Course

Jeremy Howard's top-down approach to ML education complements the bottom-up approach we take at ML0X. The fast.ai library provides high-level abstractions that make it quick to get results, while the free course explains the theory behind each component.
